# Examples Data Files

This folder contains various data files for use with PixelAudio example sketches. 
Some of the subfolders contain files that are intended for special purposes.
- The JSON folder contains .json files intended for use with the WaveSynth examples. 
- The Bag, Body, and Sessions-Bagatelle folders are intended for use with Bagatelle presets, but also contain audio and image files you can load in most sketches. 
- Bag contains a folder of brushes that you can load in Bagatelle using then 'o' key command.
- Body contains a folder, session_02, that contains brush and audio configuration data using then 'o' key command.
- Sesssion-Bagatelle contains brush and audio configuration data using then 'o' key command. 
- The image and audio files can be opened by most of the example sketches.
